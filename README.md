# Organ Health Analysis

Cleaned Python code and small notebook wrappers for organ-health analyses using UK Biobank data.

The real analysis code is in `src/`.

The notebooks in `notebooks/` are simple readable launchers for the code.

UK Biobank data and generated results are not included. Keep private files local in `data/` and `results/`.

Included analyses:

- organ age-gap models
- PC1/PCA organ metrics
- PLS organ metrics
- brain MRI structure associations
- PRS associations
- diagnosis/logistic regression analyses
- mediation analyses
